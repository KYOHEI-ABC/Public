# Blockbench
- 形式 glTF
  - Embed Textures
- 上の層のパーツ (アーマー) は使わないなら削除しておくこと
- 顔だけ解像度を高くできる
  - 顔パーツを右クリック
  - UV Mode
  - Pre-face UV
  - 左のウィンドウでマッピングの移動、サイズを拡大できる

# Godot
- プロジェクトのフォルダに glTF ファイルを持ってくると、テクスチャ (png) も生成される
- シーン内にドラッグ、ドロップすれば OK
- インポートしたモデルを直接編集はできない
- モデルを新しいシーンとして保存する必要がある
